# npx-beom-gyu-choi

> **Software Engineer (AI / Full-Stack)** > "Bridging the gap between advanced LLM architectures and scalable web applications."

이 프로젝트는 터미널(CLI) 환경에서 인터랙티브하게 확인할 수 있는 **최범규(BEOMGYU CHOI)**의 터미널 명함 및 미니 이력서 패키지입니다. 
양산형 AI 템플릿 느낌이 아닌, 핵심 연구 이력과 대회 수상 데이터를 깔끔한 표(Table) 구조로 시각화했습니다.

---

## Run Directly

내 컴퓨터에 별도의 설치 없이, 터미널창(Mac, Linux, Windows CMD/PowerShell)을 열고 아래 명령어를 입력하면 즉시 실행됩니다.

```bash
npx npx-beom-gyu-choi
